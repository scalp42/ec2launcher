ec2launcher

Shared scripts to launch EC2 instances

Download this project as a .zip file Download this project as a tar.gz file

Purpose

ec2launcher is a tool to help launch Amazon EC2 instances by providing concepts of environments and applications. An environment describes a collection of instances that share common settings. Typical environments might include development, staging and production. In contrast, an application represents a full application stack running on an EC2 instance. Launching a new instance combines the configurations from both an environment and an application.

Setup

Install the ec2launcher gem:

gem install ec2launcher

Create a new workspace for environments and applications:

ec2launcher init myworkspace

This creates a directory with the new myworkspace that contains a config.rb, an environments directory and an applications directory.
